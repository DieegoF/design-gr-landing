# Design.gr - Landing Page

Landing page para Design.gr, carpintería de muebles a medida en La Plata, CABA y AMBA.

## Tech Stack

- React 19 + TypeScript
- TanStack Start + TanStack Router
- Tailwind CSS 4
- shadcn/ui components
- Vite 8

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

### Vercel
Import this repo in [vercel.com/new](https://vercel.com/new). Framework is auto-detected.

### Railway
Import this repo in [railway.com/new](https://railway.com/new). Set build command to `npm run build`.
